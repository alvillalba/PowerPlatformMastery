# PL-200: Microsoft Power Platform Functional Consultant

## Overview

**Exam Duration**: 120 minutes  
**Passing Score**: 700/1000  
**Exam Type**: Processed (not open book)

> **Study Strategy**: Use this repo as your roadmap and checklist. Study the actual content on [Microsoft Learn](https://learn.microsoft.com/training/powerplatform/power-platform-functional-consultant).

## Exam Objectives

The PL-200 certification validates your ability to configure Power Platform solutions, analyze requirements, and implement solutions that meet business needs.

## Exam Topics

### 1. Configure Microsoft Dataverse (20-25%)

#### Dataverse Configuration
- [ ] Create and manage entities
- [ ] Configure entity relationships
- [ ] Configure security roles and field-level security
- [ ] Configure business rules
- [ ] Configure workflows
- [ ] Manage data

#### Data Management
- [ ] Import and export data
- [ ] Configure data validation
- [ ] Manage duplicate detection
- [ ] Configure auditing

### 2. Configure Power Apps (25-30%)

#### Canvas Apps
- [ ] Design app architecture
- [ ] Connect to data sources
- [ ] Implement formulas and controls
- [ ] Configure app sharing and permissions

#### Model-driven Apps
- [ ] Customize forms and views
- [ ] Configure business rules
- [ ] Configure dashboards
- [ ] Configure app navigation

### 3. Configure Power Automate (20-25%)

#### Cloud Flows
- [ ] Create automated flows
- [ ] Create instant flows
- [ ] Create scheduled flows
- [ ] Configure connectors
- [ ] Implement error handling

#### Business Process Flows
- [ ] Design business process flows
- [ ] Configure stages and steps
- [ ] Configure branching logic

### 4. Implement Power Virtual Agents (10-15%)

#### Chatbot Configuration
- [ ] Create chatbots
- [ ] Configure topics and conversations
- [ ] Integrate with Power Automate
- [ ] Publish chatbots

### 5. Manage Solutions and Environments (15-20%)

#### Solution Management
- [ ] Create and manage solutions
- [ ] Export and import solutions
- [ ] Manage solution dependencies
- [ ] Version control

#### Environment Management
- [ ] Configure environments
- [ ] Manage environment security
- [ ] Configure data policies

## Study Tips

1. **Focus on configuration, not development** - This exam is about configuring solutions, not writing code
2. **Hands-on practice is essential** - Configure actual solutions in a Developer environment
3. **Understand business requirements** - This role bridges business and technical
4. **Know solution management** - Solution lifecycle is a key topic
5. **Practice with different data sources** - Understand various connector types
6. **Manage time** - 120 minutes for the exam, practice pacing
