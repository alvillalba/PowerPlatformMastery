## Key Concepts

### Business Value (15-20%)
- Power Apps value: Low-code app development
- Power Automate value: Process automation
- Power Pages value: Customer self-service portals
- Dataverse value: Unified data platform
- Connectors value: Integration capabilities
- ROI and cost reduction scenarios

### Environment Administration (15-20%)
- Environment types: Production, Sandbox, Developer
- Power Platform Admin Center navigation
- DLP (Data Loss Prevention) policies
- User and security role management
- Environment capacity and limits
- Governance and compliance

### Power Apps (25-30%)
- **Canvas Apps**: Blank canvas, flexible design, any data source
- **Model-driven Apps**: Data-first, Dataverse, structured
- When to use Canvas vs Model-driven
- Common controls and responsive design
- Data connectors (SharePoint, Excel, Dataverse, SQL)
- Sharing and permissions

### Power Automate (15-20%)
- **Cloud Flows**: Automated, Instant (button), Scheduled
- **Desktop Flows**: RPA for desktop automation
- Triggers and actions
- Connectors (standard vs premium)
- Process mining
- Loops and branches

### Power Pages (10-15%)
- Customer self-service portals
- External website creation
- Authentication methods
- Design Studio areas
- Security concepts
